Listas desordenadas

La etiqueta <ul> define una lista desordenada. Entre la etiqueta de apertura y la de cierre se incluyen los elementos de la lista con la etiqueta <li>.

<ul><li><Patatita/li> <li>Patatita2</li> <li>Patatita3</li> </ul>
Resultado de la lista

  • Patatita
  • Patatita2
  • Patatita3

Estilo a las listas Por defecto, delante de cada elemento se pone un disco o punto negro. Podemos cambiar este marcador (o viñeta) modificando el atributo list-style-type con hojas de estilo. Para listas desordenadas, los valores más usuales son:

none: No muestra ningún marcador.
disc: Muestra un punto negro. Es el valor por defecto.
circle: Un círculo vacío.
square: Un cuadrado.

Listas ordenadas

La etiqueta <ol> define una lista desordenada. Entre la etiqueta de apertura y la de cierre se incluyen los elementos de la lista con la etiqueta <li>.

<ol><li><Patatita/li> <li>Patatita2</li> <li>Patatita3</li> </ol>
Resultado de la lista

  1. Patatita
  2. Patatita2
  3. Patatita3

Al igual que en las lista desordenadas, podemos cambiar los marcadores de la lista, a continuación están los más utilizados:

none: No muestra ningún marcador.
decimal: Números decimales: 1, 2, 3... 10, 11...
decimal-leading-zero: Números decimales con zeros delante: 01, 02, 03... 10, 11...
lower-roman: Números romanos en minúsculas: i, ii, iii, iv, v...
upper-roman: Númeors romanos en mayúsculas: I, II, III, IV, V...
lower-alpha: Letras en minúscula: a, b, c, d...
upper-alpha: Letras en mayúscula: A, B, C, D...
