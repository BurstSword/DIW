Posicionamientos

La propiedad position tiene seis valores posibles, que son :

Static

Este valor el elemento al que se le aplique sigue el flujo normal de la página

Codigo cajas

Le aplicamos la propiedad left y se ve que la ha ignorado.

Relative

Este valor el elemento al que se le aplique se posiciona en función del flujo normal de la página

Codigo cajas

En este caso se ve como se le ha aplicado la propiedad left y sí que le ha afectado el cambio.

Absolute

Este elemento toma el posicionamiento respecto a la ventana del navegador y no el flujo normal

Codigo cajas

Se le a aplican las propiedades y se ve como el cubo se posiciona encima de los otros dos y movido hacia la derecha.

Fixed

Este elemento funciona de manera similar al Absolute, pero este aunque hagas scroll hacia abajo seguirá tomando como referencia la ventana, así que bajará contigo en la misma posición.

Codigo cajas

Al descomentar el código podremos ver como el cubo rojo nos seguirá por toda la página

Inherit

Este elemento tomará las propiedades que tenga el padre establecidas.

Initial

Esta propiedad haría la misma función que static.

Sticky

Por último tenemos esta, que funciona como relative hasta que bajamos el scroll, entonces pasa a funcionar como si fuera fixed.
